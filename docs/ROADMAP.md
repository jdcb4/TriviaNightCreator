# Roadmap

Future ideas only. Roadmap entries are **not** action items.

Do not implement anything from this list unless the user explicitly asks for that feature or moves it into an active plan.

When the user mentions a future idea that is out of scope for the current task, add it here rather than implementing it opportunistically.

## Ideas

### Accounts and saved quizzes

- Add optional user accounts.
- Let account users return to saved quizzes.
- Let anonymous quizzes be claimed by an account, if a safe flow is later designed.
- Add a saved quiz dashboard.
- Add quiz duplication.
- Add long-term storage and account-based archive restoration.

### AI assistance

- Add AI-assisted individual question generation.
- Add AI-assisted full trivia night generation.
- Suggest answer variants and acceptable answers.
- Suggest round themes.
- Suggest tiebreakers.
- Suggest bonus-round ideas.

### Decorative asset library

- Create a reusable library of roughly 50 black-and-white quirky cartoon decorations for answer sheets.
- Add theme packs for decorations.
- Let hosts select or exclude specific decorative assets.
- Add seasonal or event-specific decorative packs.

### Handout-based special rounds

- Add custom printable handout support for specific special round types.
- Picture round: name items, people, places, or movies from images.
- Movie-still round.
- Logo identification round.
- Map round.
- Emoji clue round.
- Matching round.
- Dingbat/rebus round.
- Timeline ordering round.
- “Name these 4 movies from the pictures” round.

### Media rounds

- Add picture rounds.
- Add audio rounds.
- Add video rounds.
- Add presenter-safe media playback controls.

### Question reuse

- Add a reusable question bank.
- Add import/export of rounds.
- Add duplicate trivia night.
- Add duplicate round.
- Add favourite questions or round templates.

### Team/device features

- Add team phone submissions.
- Add automatic marking for submitted answers.
- Add live scoreboard display.
- Add multi-device presenter control.
- Add marker-specific access and score-entry flow.

### Print/export

- Add full host pack export.
- Add pre-filled team answer sheets.
- Add custom special-round handouts.
- Add richer branding templates.
- Add downloadable backup/export once requirements are clear.
